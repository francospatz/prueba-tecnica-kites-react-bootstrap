**Prueba técnica Kites**

Product visualizer made in React with React-Bootstrap

## Quick start
```
npm i 
```
```
npm start
```
## Dependencies

-    fontawesome
-    bootstrap
-    classnames
-    react
-    react-bootstrap
-    react-dom
-    react-router-dom
-    react-scripts
-    react-router-hash-link
-    sass
-    simplebar-react
-    uuid
